Titanic

Résumé : Lors de ce projet les enseignants nous on laisser à disposition une base de données composée de données récoltées sur le naufrage du Titanic. La problématique fût de trouver les facteurs ayant influencé la survie des passagers. Pour cela, nous devions modéliser puis implémenter une base de données que peuplez à partir des données mises à disposition.
